Increase VM Size

Typically, a good workflow for managing VM size is to increase the drive allocation in increments as it fills up. This requires the user to watch for symptoms of a full disk (failing app installs, etc.). This happens in two parts:

  1. Increasing the disk allocation in the host OS's virtualization app
  2. Utilizing the newly available disk space within the guest virtual machine

Increase Disk Allocation in Host App

  1. With the VM open, click Virtual Machines / Settings / Hard Disk, and adjust the disk space here

Allocate Disk Space Within Guest VM

  1. Install GParted
    2. sudo apt update && sudo apt upgrade -y && sudo apt install gparted
    bash
  2. Use GParted to allocate the newly available disk space within the guest VM. Launch via the application menu or via terminal.
