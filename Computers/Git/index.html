Index Lock File Exists

Git - fatal: Unable to create '/path/my_project/.git/index.lock': File exists

Delete the index.lock file, either through your File Explorer, or with the following command:

Stack Overflow Post 
rm -f ./.git/index.lock
bash

Note: If the file cannot be deleted, restart your PC or end all git.exe tasks in Task Manager.
