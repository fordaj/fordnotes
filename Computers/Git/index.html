Index Lock File Exists

Git - fatal: Unable to create '/path/my_project/.git/index.lock': File exists

Delete the index.lock file, either through your File Explorer, or with the following command:

Stack Overflow Post 
rm -f ./.git/index.lock
bash

Note: If the file cannot be deleted, restart your PC or end all git.exe tasks in Task Manager.

Smudge Filter LFS Failed

During a branch checkout:

error: external filter 'git-lfs filter-process' failed

  1. Delete local offenders (the repo if cloning, or local file changes if checking out a branch. Untracked files should be added prior to resetting).
  2. git lfs install --skip-smudge
    bash
  3. Run command causing smudge error (clone, checkout) 
  4. git lfs pull
    bash
  5. git lfs install --force
    bash
