# fordnotes
A Markdown Driven React App purpose-built for knowledge reusability.

# Authors
Andrew Ford and Kailey Waal

## Build Deployment
1. Run pre-start scripts to update `FileStructure.jsx` and format MDX
```bash
npm run prestart
```

2. Verify changes are correct
```bash
npm start
```

3. Deploy
```bash
npm run deploy
```
