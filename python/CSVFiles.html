CSV Files

Write to CSV File

Begin by importing the CSV library:

        import csv

The code below will write the fields and content to FILE.csv:

        fields = ['MY','HEADERS','GO','HERE']
content =   [
                {'MY':'my', 'HEADERS':'rows', 'GO':'go', 'HERE':'here'},
                {'MY':'also', 'HEADERS':'they', 'GO':'go', 'HERE':'here'}
            ]
with open('FILE.csv', 'w') as csvfile:
    writer = csv.DictWriter(csvfile, fieldnames = fields)
    writer.writeheader()
    writer.writerows(content)

Note: the 'w' instructs CSV writer to overwrite the file contents. Changing it to 'a+' will append the dictionary onto the file.
